## Description

Backend part of [Vending machine app](https://mvpmatch.notion.site/Full-stack-ac8a8b07bee84937968377c840b6fa29) 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
